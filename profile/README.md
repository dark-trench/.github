# Dark Trench

Below the surface, signals travel differently.

Dark Trench is a place for things lowered into black water: instruments, maps,
runtimes, questions, small machines with long memories.

Some return with readings. Some return changed.

At the edge of the shelf:

- [Squid Mesh](https://github.com/dark-trench/squid_mesh), a lattice for motion
  under pressure.
- [Squid Sonar](https://github.com/dark-trench/squid_sonar), a way to listen
  when the lights are gone.

We favor ballast, traces, pressure marks, recoverable paths, and systems that do
not need daylight to be understood.
