# Dark Trench

The surface keeps its weather.

Below it, other laws begin.

Dark Trench is a shelf-edge workshop for software that has to keep moving when
the light is gone: systems with memory, instruments for hidden motion, runtimes
that leave pressure marks instead of promises.

What lives here:

- [Squid Mesh](https://github.com/dark-trench/squid_mesh): a Jido-native durable
  workflow layer for Elixir, built for long-running, recoverable, human-in-the-loop
  and agentic execution.
- [Squid Sonar](https://github.com/dark-trench/squid_sonar): an embeddable runtime
  UI for Squid Mesh, built to inspect runs, surface state, and make execution
  visible while it is still in motion.

The work is quiet by design: clear state, readable traces, recoverable paths,
and machinery that can descend, listen, and come back carrying proof.
